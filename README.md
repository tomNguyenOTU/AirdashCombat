# AirdashCombat

controls (keyboard)

WASD - movement

S - fastfall (tap or hold)

spacebar/I - jump

O - dash

P - attack (hold for heavy)

--------------

controls (controller, xbox layout)

left stick - movement

left stick down - fastfall (tap or hold)

B - jump

LB/A - dash

RB/X/Y - attack (hold for heavy)
